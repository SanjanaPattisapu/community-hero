# Community Hero – AI Powered Hyperlocal Problem Solver

## 📌 Overview

Community Hero is an AI-powered web application built to help citizens report, track, and resolve hyperlocal community issues such as potholes, water leakages, damaged streetlights, garbage accumulation, and other civic infrastructure problems.

The platform uses Google's Gemini AI to intelligently categorize issues, provide AI-powered insights, and improve communication between citizens and local authorities.

## 💡 Problem Statement

Communities often face civic issues like potholes, water leakages, damaged streetlights, and waste management concerns. Existing reporting systems are fragmented and lack transparency.

Community Hero provides an AI-powered solution that enables citizens to report, monitor, and track issues efficiently while improving transparency and community participation.
